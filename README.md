# job-portal
Job Portal Spring Boot Project
A full-stack Job Portal Application built using Spring Boot, JSP/Scriptlet, MySQL, and JavaScript/CSS.
It supports job posting, applications, OTP verification, and admin management system.

✨ Features
#User Features
  User registration & login
  Email OTP verification
  Forgot password (OTP-based)
  Apply for jobs
  Prevent duplicate applications

#Admin Features
  Add / update / delete jobs
  Manage users (block / activate / delete)
  View all applications
  Search users(AJAX-based) & jobs Job-table client-side filtering

#Security Features
  OTP-based email verification
  Time-based OTP expiry (2 minutes)
  Secure password reset flow
  Basic Spring Boot Security(Login)

#Tech Stack
  Backend: Spring Boot, Spring MVC
  Frontend: JSP / HTML / CSS / JavaScript
  Database: MySQL
  ORM: Hibernate / JPA
  Email Service: Java Mail Sender
  Build Tool: Maven
